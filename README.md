## ⚡ 介绍

书斋是免费开源的epub格式的电子书阅读器，可通过在线，文件，微信公众号，邮箱等多种方式导入书籍。

- Web: [书斋-web](https://xingxia)

## 🏭 特性
- **epub**：epubjs渲染
- **react**: react前端框架开发

## 🚀 开发

```bash
# clone
git clone git@github.com:banapy/book_reader.git

# enter the project directory
cd book_reader

# install dependencies
pnpm i

# start the service
pnpm start
```
